About FileIntegrityChecker

FileIntegrityChecker is a comprehensive tool designed for Linux systems to monitor the integrity of files and directories. By calculating and comparing SHA256 hashes, this application helps you quickly detect unauthorized modifications, additions, or deletions.

Features

  • Directory Selection: Easily choose the directory you want to monitor.
  • Recursive File Scanning: Calculates SHA256 hashes for every file (reading in 4096-byte chunks) within the chosen directory.
  • Initial State Storage: Saves the initial file hashes in a JSON file for future comparisons.
  • Integrity Verification: Manually or periodically compare the current state to the saved state to detect modifications, deletions, or new files.
  • Scheduled Scanning: Automate the integrity checking process at regular intervals, with a simple start/stop mechanism.
  • Visual Notifications: Receive alerts via the GUI output and pop-up messages when changes are detected.
  • Robust Logging: All operations and errors are logged into a file (file_integrity_checker.log) for troubleshooting.
  • Easy-to-use Interface: A clean and professional GUI built using Python's Tkinter.
